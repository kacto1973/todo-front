# 📝 Task Manager - Frontend

## 🌟 Descripción

Esta es la interfaz de usuario para el proyecto **Task Manager**. Es una aplicación web moderna y ligera desarrollada con **JavaScript Vanilla**, diseñada para gestionar tareas en tiempo real conectándose a una Web API externa.

La aplicación permite visualizar, agregar y eliminar tareas de forma dinámica sin recargar el navegador, utilizando peticiones asíncronas.

---

## 📂 Estructura del Proyecto

```text
task-manager-frontend/
├── index.html    # Estructura principal y maquetado
├── style.css     # Estilos modernos y diseño responsivo
└── app.js        # Lógica de consumo de API (Fetch) y control del DOM
```
